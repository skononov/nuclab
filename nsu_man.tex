В данном пособии приводятся методические указания по выполнению пяти лабораторных работ в рамках курса 
<<Ядерный практикум>> для студентов 4-го курса физического факультете НГУ. Каждое описание лабораторной 
работы имеет: краткое введение в соответствующую область экспериментальной физики и в методику, 
применяющуюся в данной работе; перечень используемого оборудования; задания для студентов и указания
по их выполнению; форму отчета о выполненной работе; список рекомендованной литературы для подготовки 
к защите работы. Настоящее пособие переработано с учетом модернизации оборудования для лабораторных 
работ и появлением новых работ на основе закупленного оборудования, по сравнению с предыдущим 
изданием пособия.
